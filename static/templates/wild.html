Data Extraction from a PDF Document

Instructions

Task: Extract information from a PDF document

Collect the following information from the given document:

  • Document information
    • Country of author (browse the web and look for the domain of the email address when necessary)
    • Number of authors for the corresponding affiliation
  • Dataset information
    • Dataset name
    • Human (e.g., users, accounts, patients, survey participants) vs. non-human subjects (e.g., animals, tweets, medical records, images)
    • Off-the-shelf (e.g., from UCI Machine Learning Repository, Kaggle) vs. author-collected
    • URL of the dataset (if available)
  • Participant information
    • Country of participants
    • Number of participants per country

Prioritize human-subject data

If you cannot find human-subject, report the number of records and select "Non-human" in the "Subject type" field. For example, if you have a dataset collected from Twitter, report the number of users when available instead of the number of tweets.
